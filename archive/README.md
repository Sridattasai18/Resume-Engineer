# Archive

This directory contains deprecated and stray files from earlier prototypes and client-only versions of **Resume Engineer**. These files are kept here for historical reference rather than being deleted.

## Contents

- **`demo.html`**: The original standalone, client-only single-page HTML frontend prototype. It merges templates and UI widgets without backend reliance.
- **`app.js`**: The original client-only JavaScript file used by the root prototype to fetch LLM APIs directly in the browser. 
- **`styles.css`**: A copy of the application's stylesheet. Active copies are maintained separately in `static/styles.css` (for the server-backed Flask version) and `v2/styles.css` (for the V2 client-only version).
